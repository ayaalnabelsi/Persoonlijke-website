# Vrijdag reflectie Week 1
Vrijdag heb ik zelfstandig gewerkt aan mijn website vooral met animaties en vormgeving. Ik heb geëxperimenteerd met kleuren, lettertypes en vormen om alles mooi te maken. Met de klas hebben we een rondje gedaan het was leuk om te zien hoe anderen hun code hebben opgebouwd en om ideeën op te doen voor mijn eigen website. Ik merkte dat stap voor stap werken helpt om problemen op te lossen en dat ik steeds meer zelfstandig aan de slag kan.

# Wekelijksverslag Week 1
In week 1 heb ik me vooral beziggehouden met het bedenken van ideeën voor de persoonlijke pagina die we gaan maken en met het formuleren van drie leerdoelen waaraan ik gedurende deze minor wil werken. Ik heb een schets gemaakt en een eerste opzet van de pagina opgesteld. Het was best moeilijk om een idee voor mijn eigen profiel en leerdoelen te formuleren zonder tekst, maar juist dat maakte het een uitdagende week. Daarnaast vond ik het erg interessant om het werk van andere studenten te bekijken en te zien hoe zij hun ideeën en leerdoelen hebben uitgewerkt met beeld, geluid en animatie.

# Maandag reflectie Week 2
Maandag heb we een API workshop van Sanne gehad. Tijdens de workshop heb ik actief meegetypt en stap voor stap gevolgd wat er werd uitgelegd. Hierdoor kreeg ik beter inzicht in hoe een API werkt en hoe je data kunt ophalen en gebruiken in je website. Het was soms best ingewikkeld, maar door mee te typen en mee te doen kon ik het proces beter begrijpen. De rest van de dag ben ik bezig geweest met het vormgeven van de website met CSS. Ik heb gewerkt aan de layout, kleuren, lettertypes en animaties. Ik heb geprobeerd om alles visueel bij elkaar te laten passen en mijn ontwerp steeds verder te verbeteren. Door te experimenteren met verschillende stijlen kreeg mijn website steeds meer een eigen uitstraling.

# Dinsdag reflectie Week 2
Tijdens de workshop heb ik veel geleerd over het definiëren van schermen en websitegroottes met clamp. Ik heb ook geleerd hoe utility classes worden gebruikt voor consistente stijlen. Daarnaast heb ik geleerd dat je em gebruikt wanneer iets mee moet schalen, en rem wanneer een waarde vast moet blijven staan.Toch vond ik het lastig om de uitleg tijdens de workshop helemaal te volgen, omdat het vooral ging over elementen waar we nog niet eerder mee hadden gewerkt, dit zijn juist dingen die ik heel graag wil leren gebruiken en ik wil er de komende tijd extra mee oefenen. 

# Woensdag reflectie Week 2
Vandaag heb ik me verdiept in JavaScript. Ik heb veel nieuwe woorden en functies geleerd en beter begrepen hoe ze werken. Ik heb veel onderzoek gedaan en hulp gekregen van de docent over hoe ik studenten via een API kan ophalen. Het werk was best intensief, ik vond het lastig om codes en woorden goed aan elkaar te koppelen, waardoor het proces langzaam ging en ik vastliep. Tijdens de les heb ik veel geprobeerd en soms lukte iets niet meteen, maar zo leerde ik steeds bij.
Samen met Arvid heb ik code reviews gedaan. We hebben elkaars werk bekeken, de eisen doorgenomen en feedback gegeven. Dit hielp om de kwaliteit van de code te verbeteren en nieuwe ideeën op te doen.

# Wekelijksverslag reflectie Week 2
Ik vind dat ik deze week veel heb geleerd, het was een drukke maar leerzame week. Ik heb veel geleerd over code, vooral door intensief te werken met een API. Hoewel het ophalen van data soms lastig bleef heb ik er veel van geleerd. Ik heb ook veel geëxperimenteerd met animaties. Ik heb verschillende dingen uitgeprobeerd, fouten gemaakt en daarna verbeterd. Ik ben tevreden met het uiteindelijke resultaat van mijn website. Het voelt goed om te zien hoe alles samenkomt en om mijn kennis stap voor stap te vergroten. Hoewel ik nog steeds moeite heb met JavaScript wil ik me hier de komende tijd extra op focussen. Ik wil elke dag nieuwe woorden en functies leren om mijn kennis verder te ontwikkelen en meer zelfvertrouwen te krijgen in het programmeren.
